Comment retrouver le mot de passe wifi si vous l’avez oublié ?

Par: Yann kasongo 234 123

Il vous arrivera peut-être d’oublier le mot de passe de votre réseau sans fil, ce qui est bien ennuyeux. Que votre ordinateur fonctionne sous Windows ou macOS X, il est tout d’abord possible de récupérer le mot de passe en allant dans les paramètres de l’ordinateur ou sur la page de configuration du routeur. Si cela ne fonctionnait pas, il faudrait alors réinitialiser le routeur afin d’utiliser le mot de passe par défaut. Par contre, il est impossible de retrouver un tel mot de passe sur un appareil mobile.
